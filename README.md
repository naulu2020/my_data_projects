# HR Analytics Dashboard (Excel)

## Objectives
- Analyse employee data to identify trends in workforce composition, salary, tenure, and contract types.

## Data Used
- Sample HR dataset containing employee demographics, tenure, salary, department, performance rating, and job type.

## Tools
- Microsoft Excel (Pivot Tables, Formulas, Charts)

## Key Insights
- Female employees constitute 62% of the workforce, compared to 38% male.
- Average salary for females (64,651) is higher than for males (62,632).
- 72% of employees are on permanent contracts, while 28% are in contracting roles.
- Average employee tenure is close to 10 years.
- Finance and IT departments account for the highest salary expenses.

# Hospital Admission Analytics Dashboard (Power BI)

## Objectives
- Track hospital admission trends over time.
- Identify bottlenecks in the admission process.
- Support staffing, capacity planning, and budgeting decisions.

## Data Used
- Kaggle dataset containing patient demographics, admission dates, medical conditions, length of stay, billing amounts, and regions.

## Tools
- Microsoft Excel (data cleaning and preprocessing)
- Power BI (interactive dashboards)

## Key Insights
- Admissions are evenly distributed by gender.
- Majority of admissions involve patients aged 60 years and above.
- Q3 records the highest admission volume.

# Dublin Bike Real-Time Analysis (Python)

## Objectives
- Analyse real-time Dublin Bike usage across multiple stations within Dublin City.

## Data Used
- Real-time data collected from the JCDecaux API over several weeks.

## Tools
- Python (API data collection, data analysis, CSV storage)

## Key Insights
- Weekday usage is significantly higher than weekend usage.
- Peak usage occurs around 5:00 PM.
- Station usage is driven by location rather than station capacity.
- Mechanical bikes are more available than electric bikes.
