# Hospital Admission Analytics Dashboard

## Objectives  
- Track admission trends over time  
- Provide real-time insights into hospital admissions  
- Identify bottlenecks in the admission process  
- Produce staffing insights to optimize workforce allocation  
- Assess capacity planning and budgeting needs  
- Ensure consistent data reporting and regulatory compliance  

## Data Source  
- Kaggle dataset containing patient demographics, admission dates, medical conditions, length of stay, billing amounts, regions, and more.

## Tools Used  
- Microsoft Excel for data cleaning and preprocessing  
- Power BI for creating interactive visualizations and dashboards  

## Key Insights  
- Admissions are nearly equal between females (50.2%) and males (49.8%), indicating fair gender representation.  
- Admissions skew heavily toward older adults (60+ years), while children (0-17 years) represent only 10% of admissions.  
- The third quarter (Q3) recorded the highest number of admissions (14,190) out of 56,000 total patients.

## Takeaways  
- Fair admission practices with no significant gender gap observed.  
- Resource prioritization should focus on older adults, ensuring adequate staffing and services.  
- Increased staffing and bed allocation recommended in Q3 to handle peak admission volumes; further investigation into factors causing this peak is advised.

## How to Use  
- Open the Power BI dashboard file to explore interactive visualizations.
